IMPORTANTE: Para ejecutar el proyecto deben de correr Visual Studio en modo Administrador. 


1) Las credenciales ahorita para ingresar es Faby tanto en el usuario como en la contraseña

2) Dentro de la carpeta "Sistema.Requisiciones.BD" estan todos los scripts de la BD. Para que quede mas sencillo hice un backup de mi BD para que ustedes la puedan restaurar y no tengan que ejecutar todos los scripts. Si quieren aprender a restaurar la BD pueden guiarse con este video : https://www.youtube.com/watch?v=5W7pY7BKda0

3) Documentacion de Librerias Utilizadas

Bootstrap (Librería CSS) : https://getbootstrap.com/docs/5.1/getting-started/introduction/

Librería Ventanas Emergentes : https://sweetalert.js.org/

4) Todas las consultas a la BD se hacen atraves de procedimientos almacenados. 
Todos los procedimientos almacenados tienen que tener los parametros de OUTPUT @piCodigo y @pvError. Se pueden guiar con alguna procedimiento almacenado que yo ya haya realizado y este en la carpeta correspondiente dentro de "Sistema.Requisiciones.BD"
